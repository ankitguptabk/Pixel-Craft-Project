# Pixel-Craft-Project

## Overview
Pixel-Craft-Project is a web-based pixel art and animation creation tool, inspired by the popular PixelCraft application. Built using HTML5 Canvas, this project aims to provide an easy-to-use interface for creating and editing pixel art. It is designed to work offline as a Progressive Web App (PWA) and is optimized for mobile devices.

## Features
- **Drawing Tools:** Includes essential tools like Pencil, Eraser, Paint (flood fill), Line, Circle, and Ellipse.
- **GIF Creation:** Users can create animations by adding frames to a stack, each with a customizable delay.
- **Utility Tools:** Offers undo/redo functionality and a clear window tool to reset the canvas.
- **Advanced Features:** Supports importing images and converting them into pixel art.
- **Export Options:** Allows exporting artwork as PNG or GIF files.
- **Offline Support:** Can be installed as a PWA for offline use.

## Installation
1. **Clone the repository**
    ```bash
    git clone https://github.com/ankitguptabk/Pixel-Craft-Project.git
    cd Pixel-Craft-Project
    ```

2. **Open the project**
    Open `index.html` in your preferred web browser to start using the application.

## Usage
1. **Choose Canvas Dimensions**
   Select your desired canvas dimensions. The default is 16x16, but dimensions up to 128x128 are supported.

2. **Use Drawing Tools**
   Select from various tools such as Pencil, Eraser, and Paint to create your pixel art.

3. **Create Animations**
   Add frames to the frame stack to create animations. Each frame can be adjusted for duration.

4. **Export Your Art**
   Once your art or animation is complete, export it as a PNG or GIF file.
